This repository contains the Python and MATLAB code for the manuscript:

**"Opposing Effects of Substantia Nigra and Subthalamic Nucleus Deep Brain Stimulation on Novelty Recognition in Parkinson's Disease"**

*Authors: Bingxin Li, Chenguan Jiang, Yan Liu, Zhou Yang, Anbang Yao, Xiaoyu Zhang, Yuye Liu, Hutao Xie, Barbara Hollunder, Bryan A. Strange, Anchao Yang, Fangang Meng, Jianguo Zhang, Changming Wang, Ningfei Li, Lin Shi*

[Preprint](https://doi.org/10.64898/2026.06.17.26355856)

## Overview

This repository provides the complete analysis pipeline (Python and MATLAB) for the behavioral and task-fMRI data presented in our study. The codebase covers:

* **Behavioral Analysis (Python):**
  * Calculation of signal-detection theory (SDT) and reaction-time (RT) measures.
  * Mixed-effects statistical models and UPDRS-III clinical score analyses.
* **Task-fMRI Analysis (MATLAB/SPM):**
  * fMRI preprocessing and quality control (QC).
  * First-level general linear models (GLM) for old-versus-new memory conditions.
  * HRF-deconvolved ROI-to-ROI generalized psychophysiological interaction (gPPI) analyses for patients and healthy controls.
